# Rentify - Online Rental Marketplace

## Overview

Rentify is a full stack online rental marketplace where users can browse, list, and manage rental properties. The application is built with a React frontend and a Spring Boot backend, providing a modern and responsive user experience.

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript
* Axios
* React Router

### Backend

* Spring Boot
* Spring Data JPA
* Spring Web
* Maven

### Database

* MySQL

## Project Structure

```text
Fullstack-Rentify/
│
├── Frontend/
│   └── rentify/
│
└── Backend/
    └── RentiFy/
```

## Features

* User Registration and Login
* Browse Rental Properties
* Search and Filter Listings
* Property Details Page
* Add, Update, and Delete Properties
* Responsive User Interface
* REST API Integration
* MySQL Database Connectivity

## Installation

### Clone the Repository

```bash
git clone https://github.com/sawaldhananjay-eng/Rentify-Online-Rental-Maketplace.git
```

### Frontend Setup

```bash
cd Frontend/rentify
npm install
npm start
```

### Backend Setup

```bash
cd Backend/RentiFy
mvn spring-boot:run
```

## API

The frontend communicates with the Spring Boot backend using REST APIs.
